# Reddit Daily Games

A fun daily choice app inspired by Reddit’s “Would You Rather” style questions.  
Vote on a daily question, see live voting results, comment, and upvote comments!

---

## Features

- Daily rotating questions based on current date  
- Vote once per day, see percentage results instantly  
- Comment on questions and upvote other comments  
- Persistent votes and comments saved locally in your browser  
- Countdown timer showing time left until the next question resets  
- Simple, clean UI with Reddit-inspired orange theme  

---

## How to Use

1. Open the app and read the daily question.  
2. Choose one of two options by clicking a button.  
3. See the live vote results and comments section appear.  
4. Post your comment and upvote others' comments.  
5. Come back the next day for a new question and fresh votes!  

---

## Tech Stack

- HTML / CSS / JavaScript  
- LocalStorage for saving votes and comments (no backend)  
- Responsive and mobile-friendly design
